🃏 Decision Cheatsheets (When You're Stuck)

❓ Stuck on Regime?

✅ DO THIS Count bull signals. If 5 or more → BULL.
⚠️ IF CONFUSED Count bear signals. If 3 or more → BEAR.
Else → CHOPPY. Never invent a 4th regime.

💡 Tie-breaker: When bull=4 and bear=3 → CHOPPY. Safety first.

❓ Stuck on Rebalancing?

Do I rebalance today?
✅ Yes if: Regime changed OR (Mar/Jun/Sep/Dec).
❌ No if: Same regime AND not quarter-end.
🚫 Never rebalance on SIP day (7th).

How much to sell/buy?
→ Sell down to target. Buy up to target. Use sale proceeds, not new cash.

❓ Stuck on SIP?

✅ Invest on 7th. Use current regime table.
⛔ Pause Momentum & Alpha SIP only if BEAR.
✅ Always keep LowVol & Value SIP running.
❌ Do NOT sell anything during SIP day.

👉 New cash follows allocation, not portfolio drift.

❓ Stuck on a Bear Market?

🔴 Momentum & Alpha → 0% allocation, SIP paused.
🛡️ Value 70% + LowVol 30%.
💡 If Nifty down >15% from ATH → deploy lump sum in 3 parts (50% LowVol + 50% Value).
⏳ Wait for 2 months of CHOPPY before adding Momentum again.

❓ Stuck on Signal Data?

Where to get each signal fast:
• Nifty 50, 200-DMA, VIX → NSE India / Investing.com
• Breadth (Nifty200 % above 50-DMA) → Chartink / NSEbhavcopy
• Momentum DD → calculate from N500_Momentum50 price
• Alpha 3M return → Value Research / any ETF tracker
• FII flows → NSE (FII/DII monthly data)

❓ Stuck on Lump Sum?

📉 Crash (>15% from ATH): deploy 3 tranches over 6 weeks → 50% LowVol + 50% Value.
📈 Normal times: deploy immediately using current regime allocation.
🧠 Never average into Momentum during a crash.

❓ Stuck on Taxes?

💸 Once a year (March): sell ETFs with LTCG up to ₹1.25L gains.
🔁 Buy back next day (same ETFs).
🧾 This resets cost basis tax-free. Do it separately from rebalancing.
