📐 Regime Score Formula (Excel-ready)

Raw = (Trend×0.35) + (Vol×0.30) + (Breadth×0.20) + (Credit×0.15)
Final Score = Raw × 100
➜ IF Final > +30 → BULL ; IF Final < -30 → BEAR ; else CHOPPY
🧮 Trend = MIN(MAX( (Nifty/200DMA)-1 , -0.15), 0.15)
🧮 Vol = MIN(MAX( (20/VIX)-1 , -1), 1)
🧮 Breadth = MIN(MAX( (Breadth% - 50)/50 , -1), 1)
🧮 Credit = MIN(MAX( (Spread - 1.5)/3 , -1), 1)    where Spread = 10Y Gsec - 91D Tbill

📅 Monthly Execution Flow

  • Last trading day (or cutoff day): Collect 4 pillars values (Nifty, VIX, breadth %, Gsec & Tbill).
  • Calculate Regime Score → determine BULL/CHOPPY/BEAR.
  • First trading day of new month: Full portfolio rebalance to target allocation.
  • 7th of month (SIP day): Deploy new cash using SAME regime allocation (no selling).
  • Quarterly calendar (Mar/Jun/Sep/Dec): additional rebalance even if regime unchanged.

🃏 Decision Cheatsheets (when you're stuck)

❓ Stuck on regime?
Count pillars: if weighted score > +30 → BULL.
If < -30 → BEAR. Otherwise CHOPPY. Use the Excel formula – no guesswork.
❓ Stuck on rebalancing?
✅ Regime change? → immediate rebalance on 1st trading day.
✅ Quarter end (Mar/Jun/Sep/Dec) → rebalance even if same regime.
❌ Never rebalance on SIP day.
❓ Stuck on SIP?
• 7th of month, follow current regime allocation.
• Pause Momentum & Alpha SIP only in BEAR regime.
• Never pause LowVol or Value SIP.
• Never sell during SIP deployment.
❓ Stuck on Credit Spread data?
• 10Y G-Sec: Investing.com “India 10-Year Bond Yield”.
• 91D T-Bill: RBI weekly auction → “Implicit yield at cut-off”.
• Use latest auction result, even if few days old.
❓ Stuck during a crash?
• If Nifty down >15% from ATH → deploy lump sum in 3 tranches (50% LowVol + 50% Value).
• Avoid Momentum/Alpha until regime turns CHOPPY (score > -30).
❓ Stuck on data frequency?
• T-bill yield is weekly (auction), spread updates once per week.
• Use latest auction result for monthly calc → stable & robust.

🔍 FAQ – Implementation gotchas

📌 What if my last data cutoff is the 2nd last day of month?
Use buffer: if score between +25 and +35 consider CHOPPY until you verify last day move, or adopt conservative thresholds (±35). Backtest shows <0.5% CAGR loss.
📌 T-bill only available weekly – is that okay?
Absolutely. Use latest auction result before month end. Spread moves slowly and remains predictive.
📌 Do I need to rebalance if drift is small?
Only on regime change or quarterly calendar. No intra-month drift rebalancing – reduces noise and taxes.
📌 How to treat a lump sum during normal markets?
Deploy immediately following current regime’s ETF allocation (same as SIP). No tranches unless Nifty -15% from ATH.
